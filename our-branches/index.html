      Show Your Branch Hours Across Time Zones

      If your company has offices or branches in different cities, this widget shows visitors — at a glance — which branch is open right now in its own local time. Each branch is a small live clock arranged in a circle, and a sorted list below puts the open branches first, with "closes at" or "opens at" for each one. Add your branches above with their hours and working days, then copy the embed code into your site.

      It runs entirely in the visitor's browser using each city's time zone, so it needs no server, no API key, and no maintenance — and it updates itself as the day moves and branches open and close. The widget supports Arabic and English and a light, dark or automatic theme to match your site.

      Frequently Asked Questions

      How does it know if a branch is open?

      You set each branch's opening and closing time and its working days. The widget compares them against the current local time in that city and shows open or closed, with the next change.

      Do I need any coding?

      No. Add your branches above, then copy the embed code and paste it into your site — plain HTML, WordPress, and most builders that allow an iframe.

      Does it handle weekends and holidays?

      You choose the working days for each branch, so weekends are respected. The current version uses one opening window per day.

      Where is the data stored?

      Everything is encoded in the embed code itself — there's no database and nothing to maintain. To change branches, generate a new code.
